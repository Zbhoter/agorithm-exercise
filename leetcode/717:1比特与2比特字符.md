<https://leetcode-cn.com/problems/1-bit-and-2-bit-characters/>

# 解答

非常简单的一道题，主要熟悉 Rust 语法：

```Rust
impl Solution {
    pub fn is_one_bit_character(bits: Vec<i32>) -> bool {
        let mut i = 0;
        let mut flag = 0;
        while i < bits.len() {
          if (bits[i] == 1) {
            i = i + 2;
            flag = 1;
          } else {
            i = i + 1;
            flag = 0
          }
        }
        return flag == 0;
    }
}
```
