# 题目描述：有效数字

验证给定的字符串是否可以解释为十进制数字。


说明: 我们有意将问题陈述地比较模糊。在实现代码之前，你应当事先思考所有可能的情况。这里给出一份可能存在于有效十进制数字中的字符列表：
1. 数字 0-9
2. 指数 - "e"
3. 正/负号 - "+"/"-"
4. 小数点 - "."

当然，在输入中，这些字符的上下文也很重要。


# 解答

方法一：直接法

依据题目的要求，对字符串进行遍历，当目前字符不满足要求时，返回 false.

代码：
```cpp
class Solution {
public:
    bool isNumber(string s) {
        int index = 0, end = s.size() - 1;
        int sign_flag = 0, e_flag = 0, dot_flag = 0, num_flag = 0;
        //前方的空格
        while( index < s.size() && s[index] == ' ')     index++;
        //去除尾部的 0
        while( end > -1 && s[end] == ' ')       end--;
        
        //目前
        while( index <= end){
           // cout<<"index = "<<index<<endl;
            if( s[index] == '-' || s[index] == '+')
                if( !sign_flag && !e_flag && !num_flag && !dot_flag)
                    sign_flag++;
                else if( e_flag && sign_flag < 2)
                    sign_flag++;
                else
                    return false;
            
            else if( s[index] == 'e'){
                if( num_flag && !e_flag)
                    e_flag++;
                else
                    return false;
                
                //如果后面无数字或者为非法字符，则返回false
                if( index == end)
                    return false;
                for( int j = index + 1, flag = 0; j <= end; j++)
                    if( s[j] >= '0' && s[j]<='9')
                        ;
                    else if( !flag && (s[j] == '-' || s[j] == '+') && j != end)
                        flag++;
                    else
                        return false;
            }
  
            else if( s[index] == '.')
                if( !dot_flag && !e_flag)
                    dot_flag++;
                else
                    return false;
            
            else if( s[index] >= '0' && s[index] <= '9')
                num_flag++;
            
            else
                return false;
            
            index++;        
        }
        
        return num_flag != 0;
    }
};
```
