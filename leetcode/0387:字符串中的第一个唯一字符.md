阅读理解。

```typescript
function firstUniqChar(s:string): number {
  for( let i = 0; i < s.length; i++) {
    const it = s[i];
    if( s.indexOf(it) === s.lastIndexOf(it)) {
        return i;
    }
  }
  return -1;
}
```
