# 解答
练手题，单纯地用来熟悉 node.

```javascript
const rd = require('readline');

var rl = rd.createInterface({
    input: process.stdin,
    output: process.stdout
})

rl.on('line', ( answer) =>{
    console.log( answer.split('').reverse().join(''));
    rl.close();
})
```
