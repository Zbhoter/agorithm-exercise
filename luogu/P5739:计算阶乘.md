# 解答

其实计算集成是个很有意思的算法题。

但是，题目中给了限制，最多 12!, 不用考虑溢出，因此直接算即可。

还有，题目中给了条件，说不用循环。。

我能想到的就是打表。

另外一个想法就是用栈（递归），但实质上与循环并无区别。

```javascript
const rl = require('readline');

const rd = rl.createInterface({
  input: process.stdin,
  output: process.stdout
})

function calc(n) {
  if( n === 1) {
    return 1;
  }
  return n * calc(n-1);
}

rd.on('line',(line)=>{
  console.log( calc( parseInt(line,10)));
  rd.close();
})
```
